Staking Infrastructure

Liquid Staking · Ethereum Mainnet

SharedDeposit (sgETH)

Full-stack Ethereum liquid staking protocol. Token (sgETH + wrapped variant WSGEth), minter (SharedDepositMinterV2), rewards receiver, rollover, withdrawal queue — all auditable on-chain. Beacon chain oracle tracks validator indices and calculates virtual price for ETH rewards post-merge. Deployed and maintained through multiple upgrade cycles since 2023.

Sharedstake Oracle

Off-chain oracle for SharedDeposit V1 and V2. Reads validator indices from the beacon chain API to calculate and verify the virtual price of sgETH — the accumulating ETH rewards per share. Designed to run as a self-hosted service alongside a staking node.
